What happened to Muute?

Muute originally created to be a Youtube channel focused on Minecraft, and quickly evolved into a site built in PHP for random stuff. Since then my interest in creating a site based on a Youtube channel has declined, and I needed somewhere to show my work. I tried rewriting all the code on Muute, but it still wasn't quite right. Hence Asher Foster. A brand new identity, associated with me, and my work. Now with all the code open source, it's a perfect place for me to showcase my work, and play around with various web technologies.
